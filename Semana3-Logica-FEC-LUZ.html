Operaciones con conjuntos

  1. Unión
    La unión de dos conjuntos AAA y BBB es el conjunto A∪B={x:x∈A oˊ x∈B}A \cup B = \{ x : x \in A ~ó~ x \in B \}A∪B={x:x∈A oˊ x∈B}
    Propiedades

    • A∪A=AA \cup A = AA∪A=A y A∪∅=AA \cup \empty = AA∪∅=A, para todo conjunto AAA.
    • A⊂BA \subset BA⊂B si y sólo si A∪B=BA \cup B = BA∪B=B
    • Conmutatividad, es decir, A∪B=B∪AA \cup B = B \cup AA∪B=B∪A
    • Asociatividad, es decir, A∪(B∪C)=(A∪B)∪CA \cup (B \cup C) = (A \cup B) \cup CA∪(B∪C)=(A∪B)∪C

  2. Intersección
    La intersección de dos conjuntos AAA y BBB es el conjunto A∩B={x∈A:x∈B}A \cap B = \{ x \in A : x \in B \}A∩B={x∈A:x∈B}
    Propiedades

    • A∩A=AA \cap A = AA∩A=A y A∩∅=∅A \cap \empty = \emptyA∩∅=∅, para todo conjunto AAA
    • A⊂BA \subset BA⊂B si y solo si A∩B=AA \cap B = AA∩B=A
    • Conmutatividad: A∩B=B∩AA \cap B = B \cap AA∩B=B∩A
    • Asociatividad: A∩(B∩C)=(A∩B)∩CA \cap (B \cap C) = (A \cap B) \cap CA∩(B∩C)=(A∩B)∩C

  3. Diferencia
    La diferencia de dos conjuntos AAA y BBB es el conjunto A∖B={x∈A:x∉B}A \setminus B = \{ x \in A : x \notin B \}A∖B={x∈A:x∈/B}

  4. Diferencia simétrica
    La diferencia simétrica de dos conjuntos AAA y BBB es el conjunto AΔB=(A∖B)∪(B∖A)A \Delta B = (A \setminus B) \cup (B \setminus A)AΔB=(A∖B)∪(B∖A) Hereda las propiedades de la unión.

  5. Complemento
    El complemento de un conjunto AAA respecto a un conjunto universal UUU es el conjunto A′=U∖AA' = U \setminus AA′=U∖A
    Propiedades

    • (A′)′=A(A')' = A(A′)′=A para todo conjunto AAA
    • A∩A′=∅A \cap A' = \emptyA∩A′=∅
    • A∪A′=UA \cup A' = UA∪A′=U
