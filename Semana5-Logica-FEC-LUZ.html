Conjunto potencia

El conjunto potencia de un conjunto AAA es el conjunto de todos los subcinjuntos de AAA. Se le denota P(A)P(A)P(A).

Ejemplo

Si A={3,5}A = \{ 3,5 \}A={3,5} entonces P(A)={∅,{3},{5},{3,5}}P(A) = \{ \empty, \{3\}, \{5\} , \{3,5\}\}P(A)={∅,{3},{5},{3,5}} Observa que X⊂AX \subset AX⊂A y X∈P(A)X \in P(A)X∈P(A) significan exactamente lo mismo.

Pares ordenados y no ordenados

Par ordenado

A un conjunto de la forma {a,b}\{a,b\}{a,b} se le llama par no ordenado. Observa que {a,b}={b,a}\{a,b\} = \{b,a\}{a,b}={b,a}

Par no ordenado

Un par ordenado se define a partir de un par no ordenado de la siguiente manera:
(a,b)={{a},{a,b}}(a,b) = \{ \{a\}, \{a,b\} \}(a,b)={{a},{a,b}}
Esta definición algo artificiosa es un recurso técnico para poder distinguir el primer elemento de un par del segundo.

Producto cartesiano

El producto cartesiano de dos conjuntos AAA y BBB es el conjunto A×B={(a,b):a∈A,b∈B}A \times B = \{ (a,b) : a \in A, b \in B \}A×B={(a,b):a∈A,b∈B} de todos los pares ordenados con el primer elemento en AAA y el segundo elemento en BBB.

Ejemplo

Si A={a,b}A = \{a,b\}A={a,b} y B={1,2,3}B = \{ 1,2,3 \}B={1,2,3} entonces A×B={(a,1),(a,2),(a,3),(b,1),(b,2),(b,3)}A \times B = \{ (a,1), (a,2), (a,3), (b,1), (b,2), (b,3) \}A×B={(a,1),(a,2),(a,3),(b,1),(b,2),(b,3)}
